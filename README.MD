# Title

MindMate API

# Description

The API for the MindMate Website

# Tech Stack

Express js, 
Cheerio js

# Installation Steps

### 1. Clone the repo

```
https://github.com/anish-sanju-jess/api.git
```

### 2. Change directory

```
cd api
```

### 3. Install Dependencies

```
npm i
```

### 4. Run the main file

```
node index.js
```
